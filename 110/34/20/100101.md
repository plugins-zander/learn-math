# 定义

## 原函数

+ 设$f(x),F(x)$为定义于$I$上的函数，
  + 若对一切的$x \in I$
  + 有$F'(x)=f(x)$
  + 称$F(x)$为$f(x)$的原函数

> + 连续函数一定存在原函数，反之不对
> + 若$f(x)$有原函数$\Rightarrow$
>   + $ \exists $无数个原函数
>   + 且任意两个原函数相差常数



## 不定积分

+ 设
  + $F(x)$为$f(x)$的一个原函数
+ 则
  + $F(x)+C$为$f(x)$所有原函数
  + 称为$f(x)$的不定积分
  + 记为$\int f(x)dx=F(x)+C$
