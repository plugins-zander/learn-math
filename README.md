+ 演算长公式用行内公式
  + 为了让其左对其，katex不支持环境
+ 极限用`^_`不规范
  + 为了让其显示为标准行间公式排版，katex行间公式极限格式和行内的一样

+ 部分三角反三角不规范
  + 部分三角反三角没有加`\`,加后katex没法解释，`arccot{x}`
  + 需要添加括号`\arcsin{x}`



+ 引用行内不要
